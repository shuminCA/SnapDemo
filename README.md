## 🌸 Commands to run the application in the terminal
```
streamlit run main.py
```

## ✨ Quick install
Load OpenAI API key with “dotenv”
```
npm install dotenv
```
Install EmbedChain
```
npm install embedchain
```
Install Streamlit
```
pip install streamlit
pip install streamlit_chat
```
